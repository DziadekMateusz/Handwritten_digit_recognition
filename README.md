# Handwritten_digit_recognition
This project implements handwritten digit recognition using TensorFlow and Keras. It supports both training and evaluation modes, allowing users to either train new models (MLP and CNN) or load previously saved ones. Images are automatically loaded, preprocessed, and split into training and test sets based on the dataset structure. The CNN model is used for inference, providing digit predictions along with confidence scores, accuracy statistics, and average prediction speed. Project designed and created during smart materials and methods lessons at university.
